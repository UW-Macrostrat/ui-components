## EditableMultilineText

Component for editing text fields on data model. Works with `ModelEditor`, becomes editable when
`isEditing` is true.

### Props:

> - _**field**_: _string_ | Name of editable field.
> - _**className**_: _string_ | Classname of component.

Examples:

```jsx
function NameEditor() {
  const field = "Name";
  className = "name-editor";

  return h(EditableMultilineText, { field, className });
}
```
