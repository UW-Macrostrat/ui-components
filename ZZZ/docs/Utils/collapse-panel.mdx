A react component that renders a button that onClick opens a collapsable panel. Internal state is managed by
button click.

### Props:

> - _**children**_: _react component_ | content to be rendered inside the collapsable panel
> - _**title**_: _string_ | Text appears in the header button for the collapsable panel.
> - _**storageID**_: _int_ | unique id for local storage
> - _**headerRight**_: _any_ | Renders to the Right of the button.
